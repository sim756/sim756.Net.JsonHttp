# sim756.Net.JsonHttp

#### **JsonHttpClient**

A simplified *Generic* method to get **JSON** using HTTP & get it deserialized into the Object of specified type.

###### Example:

```c#
Sample sample = new JsonHttpClient<Sample>("http://localhost:10000/api/values").Deserialize(); 
```
